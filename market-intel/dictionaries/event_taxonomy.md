# Event taxonomy (Physical AI)

이벤트 타입 분류. 자동 파이프라인은 아래 코드를 `event_type` 필드에 사용한다.

| Code | Description |
|------|-------------|
| funding | 펀딩, 라운드, 밸류에이션 |
| product_launch | 제품·플랫폼 출시, GA |
| partnership | 파트너십, OEM, 통합 |
| regulation_safety | 규제, 안전 표준, 인증 |
| benchmark | 벤치마크, 경연, 리더보드 |
| deployment | 배포 규모, 고객 공개, 파일럿 |
| hiring_org | 채용, 조직 개편, 인수 |
| supply_chain | 공급망, 제조, 부품 |
| research_paper | 논문, arXiv, 연구 발표 |
| patent | 특허 공개·등록 |
| simulation | 시뮬레이션, 디지털 트윈, sim-to-real |
| other | 기타 |
